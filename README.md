# Mart-Delivery-Service

KB IT's Your Life: mini project

## 주제

마트 배달 서비스 구현

## 목표

1. 객체지향 설계에 익숙해진다.
2. 객체 간의 관계에 신경쓰면서 데이터베이스 모델링을 진행한다.
3. 서비스에 대해 CRUD와 특별한 기능을 구현한다.
4. Github를 이용한 협업을 체험한다.
